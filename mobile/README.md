# OFTA Mobile App

Next.js 15 + Capacitor 7 mobile app for One for the Ages.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit: http://localhost:3000

## Build for Mobile

### First Time Setup

```bash
npm run mobile:setup
```

### iOS

```bash
npm run mobile:ios
```

### Android

```bash
npm run mobile:android
```

## Build Scripts

- `./build_playstore.sh` - Build for Google Play Store
- `./build_testflight.sh` - Build for Apple TestFlight
